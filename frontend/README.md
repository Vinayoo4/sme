# Frontend

React + Vite frontend for the SME Sync Platform.

## Environment

Copy `.env.example` to `.env` and set:

```bash
VITE_API_URL=http://localhost:4000
```

## Commands

```bash
npm install
npm run dev
npm run build
npm run lint
npm test
```

## Pages

- `/login`
- `/dashboard`
- `/feedback`
- `/inventory`
- `/notifications`
